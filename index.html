Space Hunter (2018)

Realized during my studies at Epitech, this project aimed to make a reproduction of the game "Duck Hunt" in C/CSFML. This project allowed me to understand the importance of clean code as well as the basics of video game / engine development such as the main loop.

Tower Defense (2018)

Realized as part of my schooling at Epitech, this project aimed to make a small tower defense in C/CSFML. This project allowed me to understand the importance of memory management and allowed me to improve in the way I work.

RPG (2018)

Realized during my schooling at Epitech, this project aimed to make a small RPG in C/CSFML. This project made me understand the importance of good teamwork on larger projects in a limited time. The game features several simple quests, 3 biomes with different monsters, a stats, leveling and looting system (with three different classes of equipment), etc...

Sokoban (2018)

Realized during my schooling at Epitech, this project aimed to make a terminal reproduction of the game Sokoban in C/ncurses.

Navy (2019)

Realized during my schooling at Epitech, this project aimed to make a terminal reproduction of the game Battleship in C.
The game had to be playable in "local multiplayer" between two instances of the game through the use of signals.

Tetris (2019)

Realized during my schooling at Epitech, this project aimed to make a terminal reproduction of the game Tetris.
